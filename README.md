# Photo Portfolio #

## 1. Getting Started ##
###1.1. Installing requirements ###

* PHP 7.0 +
* MySQL 5.6 +
* Symfony Framework 3.3 +
* NPM 4.1.2 +
* Bower 1.8.2 +

###1.2 Install dependencies ###
    composer install
    npm install
    bower install ./vendor/sonata-project/admin-bundle/bower.json
    bin/console assets:install
